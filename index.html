MobileNet Tensorflow.js Example - Web and Artificial Intelligence 3

Awaiting TF.js load

  1. First enable webcam and allow access when prompted.
  2. Now find an object, point camera, click and hold gather class 1 data button.
  3. Repeat for class 2 with a different object of interest.
  4. Click train and predict and wait while the model is trained live in your browser.
  5. Once trained you will see live predictions appear above the video for what it thinks it sees, either class 1 or class 2 with a confidence of prediction.
